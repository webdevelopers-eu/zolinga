This folder is automatically filled with links to skill folders from modules that provide them during the installation/update process. It is not meant to be edited manually.

For every folder like `modules/{module}/skills/*` or `system/skills/*` that contains a `SKILL.md` file, a symlink will appear here as `.agents/skills/{module}-{skill-name}`.
You do not need to add anything manually—just put your skill in the right place and it will show up here after install or update.
